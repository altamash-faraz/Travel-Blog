## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Basic understanding of HTML, CSS, and JavaScript (for customization)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/altamash-faraz/Travel-Blog.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Travel-Blog
   ```

3. **Open in your preferred code editor**
   ```bash
   code .
   ```

4. **Launch with Live Server**
   - Open `index.html` in your browser, or
   - Use a local development server like Live Server (VS Code extension)

### Quick Start
Simply download the files and open `index.html` in your web browser. The site is ready to use!

## ⚙️ Customization

### Content Updates
1. **Personal Information**: Edit the HTML files to update name, bio, and contact details
2. **Images**: Replace `background.jpg` and `venice.jpg` with your own travel photos
3. **Blog Posts**: Modify the blog content in `blog.html` with your own stories
4. **Social Links**: Update social media links in all HTML files

### Styling
1. **Colors**: Modify CSS custom properties in `style.css` for color scheme changes
2. **Fonts**: Update Google Fonts links in HTML files for different typography
3. **Layout**: Adjust CSS Grid and Flexbox properties for layout modifications
4. **Animations**: Customize animation timings and effects in CSS and JavaScript

### Functionality
1. **Form Handling**: Update form action URLs in contact and newsletter forms
2. **Blog Filtering**: Modify categories and filter logic in JavaScript
3. **Animations**: Customize scroll effects and loading animations
4. **Analytics**: Add Google Analytics or other tracking codes
